## 全局检查清单

### 每页上线前
- [ ] canonical 标签指向自身
- [ ] meta description 独立撰写（不 spin）
- [ ] Open Graph / Twitter Card
- [ ] JSON-LD 结构化数据（WebPage + 对应 Schema）
- [ ] 页面通过 Lighthouse（Performance > 90, SEO > 95）
- [ ] 移动端适配验证

### 每页上线后
- [ ] 提交到 Google Search Console
- [ ] IndexNow 推送
- [ ] 更新 sitemap
- [ ] 添加站内链接（从已上线页面链过来）
- [ ] 记录上线时间到本表

### 每周
- [ ] 检查 Search Console 收录状态
- [ ] 检查新页面是否出现在索引中
- [ ] 记录收录时间到本表

### 每月
- [ ] 检查目标关键词排名变化
- [ ] 复盘内容策略，根据实际数据调整
- [ ] 更新 SITE-STRATEGY 优先级
